# FreeMind Vision

Plateforme internationale de streaming vidéo pour créateurs, avec un focus sur l'Afrique.

## Déploiement

- **Développement**: Replit (https://replit.com)
- **Production**: Render (https://freemindvision-app-2.onrender.com)

## Technologies

- React + TypeScript (Frontend)
- Express + PostgreSQL (Backend)  
- Stripe (Paiements)
- 7 langues supportées (Français par défaut)

## Fonctionnalités

- Flux vidéo vertical style TikTok
- Monétisation 60/40 (faveur créateurs)
- Dashboard analytique
- Système de badges et achievements
- Programme de parrainage
- Achat d'actions ($108/action)
